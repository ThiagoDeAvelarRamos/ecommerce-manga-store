# Manga Store

![Status do Projeto](https://img.shields.io/badge/status-em%20desenvolvimento-orange)
![Licença](https://img.shields.io/badge/license-MIT-blue)

A **Manga Store** é uma plataforma de e-commerce focada na cultura pop asiática, oferecendo um catálogo especializado de **Mangás, Manhwas e Light Novels**. O projeto visa proporcionar uma experiência de navegação fluida e imersiva para colecionadores e leitores, com foco em design moderno e facilidade de acesso às obras favoritas.

---

## O que é?

Este projeto é uma loja virtual (E-commerce) especializada em publicações orientais. Ele foi concebido para ser um ponto de encontro entre leitores e suas obras favoritas, apresentando uma interface visualmente rica que destaca a arte das capas e facilita a descoberta de novos títulos através de categorias bem definidas e vitrines estratégicas.

## Tecnologias Aplicadas

O projeto utiliza as tecnologias web fundamentais com foco em performance e organização:

*   **HTML5 Semântico:** Uso rigoroso de tags semânticas (`<header>`, `<main>`, `<section>`, `<article>`, `<footer>`) para melhor SEO e acessibilidade.
*   **CSS3 Avançado:** 
    *   **Variáveis CSS (Custom Properties):** Centralização de cores, fontes e espaçamentos para fácil manutenção.
    *   **Layout Moderno:** Implementação de layouts flexíveis e robustos com **Flexbox** e **CSS Grid**.
    *   **Google Fonts:** Integração das fontes *Ubuntu* e *Roboto* para uma tipografia legível e moderna.
*   **SVG:** Uso de ícones em formato vetorial para garantir nitidez e leveza em qualquer resolução.

## Conceitos Usados

Para garantir um código escalável e de qualidade, foram aplicados os seguintes conceitos:

*   **Arquitetura de CSS Modular:** Organização do estilo em múltiplos arquivos (`base.css`, `layout.css`, `variables.css` e componentes específicos), seguindo princípios que facilitam a escalabilidade do projeto.
*   **Design Responsivo (Mobile-First):** Estrutura desenvolvida para se adaptar perfeitamente a dispositivos móveis e desktops, incluindo componentes adaptáveis como o menu hambúrguer.
*   **Acessibilidade (A11y):** Implementação de atributos `aria-label`, textos alternativos (`alt`) em imagens e navegação estruturada para leitores de tela.
*   **UI/UX focado em Conversão:** Design pensado na jornada do usuário, com seções de *Hero*, *Categorias em Destaque* e *Grids de Produtos* estrategicamente posicionados.
